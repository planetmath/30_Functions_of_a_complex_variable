factorization theorem for H∞ functions


Let H∞ denote the bounded analytic functions on the unit disc.

Theorem.

Every f∈H∞ can be written as

f⁢(z)=α⁢I⁢(z)⁢F⁢(z)

where |α|=1, I is an inner function and F is a bounded outer function. Conversely any function which can be so written is bounded.
